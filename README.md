# ⏱ Controle de Horas Pro - Marcos

Sistema moderno e responsivo para controle de horas extras e cálculo de lucro em tempo real.

## 🚀 Funcionalidades
- **Dashboard Financeiro**: Veja horas 50%, 100% e lucro total.
- **Timeline de Ganhos**: Histórico com visual de calendário.
- **Gestão Completa**: Adicionar, Editar, Ver detalhes e Excluir registros.
- **Configuração Salarial**: Cálculo automático de valor/hora e diária.
- **Relatório PDF**: Gere relatórios profissionais com um clique.
- **Offline First**: Seus dados ficam salvos com segurança no seu navegador.

## 🛠 Tecnologias
- HTML5 / CSS3 (Design Moderno & Responsivo)
- JavaScript (Lógica e Cálculos)
- LocalStorage (Persistência de Dados)
- html2pdf.js (Exportação de relatórios)
